# New Logo
In this question, we want to draw a diamond, with given diameter and thickness.

The inputs are:

    - Int n: which is used to calculate the diameter (diameter=n*2+1)
    - Int k: that is the thickness of out diamond
  




## Examples:

### input:
    2 4
### output:
```
    *
   ***
  ** **
 **   **
**     ** 
 **   **  
  ** **
   ***    
    *
```
